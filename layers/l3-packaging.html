LAYER 3 / 縦階層

先端パッケージ・チップレット統合

ダイをどう繋ぎ、どの基盤に載せるか。トランジスタ微細化の鈍化を補うAIスケーリングの主戦場。

投資判断の要点 ここはAIアクセラレータ供給の最大ボトルネック。トランジスタ微細化(L2)が鈍化する中、性能スケーリングの主役が「いかに多くのダイとHBMを1パッケージに高密度集積するか」へ移った。CoWoSの生産能力が事実上GPU出荷量の上限を決め、ガラス基板が次の構造変化点になる。 fact CoWoS需給 / spec ガラス採用時期

このレイヤーの二系統

L3は物理的に異なる二つの系統に分かれる。混同しやすいが、技術スタックも投資プレイヤーも別物。

系統役割代表技術接合の鍵
垂直積層系 (3D)ダイをZ方向に重ね、配線距離を最短化SoIC / Foveros / UCIe-3Dハイブリッドボンディング
水平配線基盤系 (2.5D / 基板)ダイを平面で繋ぐ配線層と、それを載せる基板CoWoS / EMIB / ガラスコア基板 / ガラスインターポーザインターポーザ・TGV

断面で理解する物理的実装

「どの部材がどの層にいるか」は断面でしか伝わらない。基準図では、CCL(基板の芯)とABF(その上下のビルドアップ絶縁膜)の上下関係、そしてMLCCが配線層ではなく裏面(ダイ直下=LSC)に載る受動部品であることに注目。各部材にカーソルを合わせると解説と部材カードへのリンクが出る。

系統A: 垂直積層 (3D)

ダイを縦に積む。鍵はハイブリッドボンディング——はんだバンプを介さず銅-銅を直接接合し、マイクロバンプを大きく上回る配線密度・帯域を得る。ピッチ競争(TSMC 6μm量産 → 3μm目標、Intel サブ5μm)がこの系統の主戦場。

系統B: 水平配線基盤 (2.5D / 基板)

ダイを平面で繋ぐ。インターポーザ(配線の盤)と基板(それを載せる土台)の二段で考える。ここにガラスが二方向から侵入する——基板コアのCCL置換としてと、Siインターポーザの置換として。タイムラインも投資妙味も別なので分けて見る。

基板の『素材の素材』(見落としがちな川上) 基板の最上流に、独占×完売の素材が潜む。CCLはさらに3素材に分解できる: T-glass(ガラスクロス)(補強繊維、コスト19%、Nittobo ~90%独占で2026最大級のボトルネック)、HVLP銅箔(導体、42%)、樹脂(26%)。stance T-glassはNANDと同じ「演算しない地味な川上」だが、独占純度はABFフィルム級。基板を語るならフィルム(味の素)とガラスクロス(Nittobo)の二つの日本企業の独占を見るべき。

AWS Trainium

Trainium3 / Trainium4 / Inferentia (Annapurna Labs) — 単位最大の商用ASICフリート

AWSが2015年買収のAnnapurna Labsを通じて自社開発する学習/推論ASIC。「商用展開されたASIC」としては最大の単位数を誇り、Trainium2が累計100万チップ近く出荷されるなど、TPUと並ぶ「もう一つのハイパースケーラASIC本命」。戦略軸はNVIDIA比50%安のコスト破壊で、Anthropicが「Project Rainier」として最大100万Trainiumを確保(2024年に$25Bの出資契約と引き換え)、Bedrockを通じてCerebrasとprefill/decodeを分担するアーキ柔軟性も特徴。2025年12月のre:InventでTrainium3が正式発表(TSMC N3 / 2.52 PFLOPS FP8 / 144GB HBM3e / 4.9 TB/s)、2026年中にUltraServer形態が一般提供される。Trainium4ではNVIDIAのNVLink Fusion技術を統合するという業界初の「NVIDIA-AWS連携」も発表され、垂直統合の覇権争いに新しい構図を持ち込んだ。

アーキテクチャ要点 — 「コスト破壊+UltraServer」

Trainium3の本質は単体スペック競争ではなく、(1) NVIDIA比~50%安、(2) UltraServer単位の物理パッケージ化、(3) Neuron SDK / 主要フレームワーク統合の3点。UltraServerはTrainium chipを16-144個まとめて単一の物理ユニットとして提供する形態で、Trainium2の64-chipからTrn3で144 chipへ拡張、Trainium4ではさらに広い"socket"へ拡張予定。AnthropicのProject RainierはTrainium2を中心に既に100万チップ規模を運用し、Trainium3への移行が進行中。

システムレベル — Trn3 UltraServer + Project Rainier

Trn3 UltraServerは144 chip / FP8 363 PFLOPS / HBM 20.7 TB / 706 TB/sを単一ユニットとして提供。AnthropicのProject Rainier(インディアナ州、複数キャンパス)はTrainiumベースの1GW級学習基盤で、Claude Opus 5以降のフロンティアモデル学習に使われる。AWSはBedrockプラットフォーム上で「prefill = Trainium / decode = Cerebras WSE-3」のような非対称分散もユーザに提示しており、ASICが単独で全てを賄うのではなく「タスク特性に合わせて互換性を保ちながら最適チップに振り分ける」モデルを実装。

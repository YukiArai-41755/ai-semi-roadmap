InfiniBand

InfiniBand (NDR 400G / XDR 800G) + Spectrum-X Photonics

量産/主流

スーパーコンピュータ由来の超低レイテンシ・クレジットベースのロスレス・ネットワーク。生成AI 初期 (2022-2024) には GPT/Llama/Claude を含むほぼ全ての先端モデル学習クラスタを押さえた事実上の標準。NVIDIA Mellanox資産を背景に、性能リーダーとしての地位を10年超維持。1ホップレイテンシ ~600ns、SHARP (Scalable Hierarchical Aggregation and Reduction Protocol) によるin-network集合通信、Adaptive RoutingなどのAI最適機能で、性能だけ見れば依然トップだが、(1) ベンダーロック (NVIDIA一択)、(2) 10万XPU超でのスケール上の運用課題、(3) Ethernet/UEC との価格差 ~2倍、(4) ハイパースケーラのオープン化要求、により2025-2026年に「学習特化のニッチ」へ縮退する転換点に立つ。NVIDIAは並行してSpectrum-X (独自 Ethernet版) で両正面戦略を取り、IB依存リスクをヘッジ。

InfiniBand 残存価値 — どこで生き残るか

ユースケースIB が残る理由
事前学習 (Pre-training) 大規模クラスタSHARP/Adaptive Routing による集合通信効率、確定的ロスレス。学習時間が直接コストに直結する場合の差が大きい
HPC / 科学計算従来から IB が標準、運用コスト・既存ソフトウェアスタックの慣性
NVIDIA DGX/HGX SuperPODNVIDIA リファレンスアーキテクチャ、検証済み・サポート保証
NVIDIA Cloud Provider (NCP) 認定OracleCloud GPU、CoreWeave、Lambda 等の認定基盤要件
金融 HFT 一部、医療画像レイテンシ要件特化、Ethernet では追いつかない用途

推論 推論 (inference) ワークロードは集合通信より個別リクエストルーティングが支配的で IB の優位が薄い。Etyhernet/UEC の方がスケール経済とエコシステムで上回るため、「学習=IB / 推論=Ethernet」の住み分けが2026-2027に明瞭化。NVIDIAも Spectrum-X を Ethernet 側 ($10B run-rate) で大きく成長させており、IB単独のNVIDIA社内優先順位は相対的に低下。

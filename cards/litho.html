リソグラフィ (EUV/High-NA)

EUV 0.33NA / High-NA EUV 0.55NA / Mask Stitching / ILT

量産/主流

AI半導体の解像度を律速する最上流。波長13.5nmのEUV(極端紫外線)を多層膜反射鏡で集光し、レチクル(マスク)から1/4縮小投影してウェハに回路を焼く。ASMLが EUVスキャナ(NXE:3800E、EXE:5000/5200)を世界独占し、装置1台あたり EUV ~USD 1.8億、High-NA ~USD 3.6-4.0億。0.33NAのレチクル上限26×33mm (~858mm²)がAI巨大ダイの物理制約の根源で、High-NA(0.55NA、アナモルフィック光学)では露光フィールドが半分(429mm²)に減るため、マスクスティッチング前提のRDL設計と CoWoS-L世代との整合が新たに要求される。Intel(14Aで採用先行)とTSMC(A12/2029まで採用見送り)で戦略が真っ二つに割れている。

確立事実 High-NA(0.55NA)はアナモルフィック光学(4×8倍縮小)を採用し、ウェハ上の露光フィールドが26×16.5mm = 429mm²に半減。AI巨大ダイは2枚のマスクをステッチングする必要があり、CoWoS設計との整合が新たな制約に。確立事実 ASML CEO Christophe Fouquetは2026年5月時点で「数ヶ月以内に High-NAでロジック・メモリ初製品が量産される」と発言、Intel 14A pilot lineが先行。

EUV光学の原理 — なぜ難しいか

光源CO2レーザでSn液滴(直径25μm)を秒5万回プラズマ化、波長13.5nm光を発生。Cymer (ASML子会社) が独占。出力500W → 600W → 1000W級へ拡張中
光学系全反射光学(透過レンズなし)。Zeiss SMT の多層膜ミラー10数枚で集光・縮小投影。1枚でも欠陥があるとシステム停止
真空全光路を真空 (EUVは大気で吸収)。ウェハ・マスクは静電チャック
マスク反射型 (Mo/Si多層膜 + TaBN吸収体)。EUVマスクカードで詳述
レジストCARからMORへ移行。EUVフォトレジストカードで詳述
マスクスティッチングレチクル上限超のダイを2-4枚マスクで継ぐ。RDL設計でつなぎ目を吸収。CoWoS-Lで常用化、Cerebras WSE-3はウェハスケール極致
ILT / CurvilinearInverse Lithography Technologyで逆問題を解き、マスクパターンが曲線化、ショット数が爆発 → MBM(マルチビーム描画)必須

AI server / 先端ノードでの位置づけ

AI GPU(NVIDIA Blackwell/Rubin、AMD MI400/MI500)、HBMベースダイ、SerDes ASIC、CPOスイッチ用シリコン、CPU(Granite Rapids、EPYC Turin/Venice)は全て N3 → N2 → A16/A14/18A の道を急ぐ。EUVスキャナの導入順=AIチップの市場投入順になるため、ASMLの装置出荷がそのままAI供給の上限を決める。

2社の戦略分岐 — Intel先行 vs TSMC見送り

軸Intel (先行)TSMC (見送り)
High-NA採用ノード14A (2026-2027 pilot, 2027 ramp)A12 (2029) 検討、A14までは0.33NAで対応
装置導入EXE:5200B Oregon設置済、商用第1号機研究機としてEXE:5000を1-2台保有、量産導入は2028以降
論拠露光数削減で歩留り改善、ファウンドリ事業再起のキーテクノロジ$400M/台のCAPEX回収困難、マルチパターニング 0.33NAでA14まで対応可
競争優位想定6-12ヶ月の先行で14Aを早期立ち上げN2 → A16 → A14とノード刻みで安全に進む
リスクHigh-NA歩留り立ち上げに失敗すれば14A遅延A14で性能差が顕在化、A12時点でHigh-NAでキャッチアップ可能か

確立事実 ASML CEOによれば High-NA EUVマシンの$360-400Mというコストとマスクの2倍化(High-NAペア+追加コスト)が、TSMC見送りの主因。推論 Intelは「装置先行投資+ノード性能差別化」で破綻リスクと引き換えにファウンドリ事業の再起を賭ける。TSMCは「マルチパターニング+SPR+CoWoS進化」の組み合わせで性能差を埋めに来る。AI顧客(NVIDIA/AMD/Broadcom)はN2/A16/A14のTSMC側に集中しており、Intel 14Aの商業成功は外部顧客獲得が条件。

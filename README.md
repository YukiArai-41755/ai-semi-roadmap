# AI半導体・データセンター技術ロードマップ — L3リファレンス実装

## 開き方
`index.html` をブラウザで直接開く(ダブルクリック)。サーバ不要。`file://` で全機能(ホバー解説・3ビュー・カード遷移)が動作する。データは `assets/graph-data.js` にインライン化済みのため CORS 制約を受けない。

## 設計の核
**全ページ・全図版・全カードは単一の `data/graph.json` から生成される。**
ノードを1つ足せば、3つの俯瞰ビュー(ツリー/グラフ/スイムレーン)・該当レイヤーページ・個別カードすべてに自動反映される。淘汰フラグ `status: "deprecated"` も一箇所で管理。

これが「網羅性」と「保守性」「遡及修正」の両立解。後続レイヤーのリサーチで先行レイヤーの修正が必要になっても、JSONを直して再生成するだけで全体に伝播する。

## ノードモデル
2種類のノード:
- **technデ** (`kind:"tech"`): 投資メトリクス(maturity/bottleneck/investment 0-5)、players、alts、status、3期間マイルストーン。
- **material** (`kind:"material"`): role/etype/props/suppliers/alts。断面図のホバーと個別カードの双方からアクセス可能。

エッジ4種: `hier`(親子) / `cross`(影響・依存) / `time`(世代交代・置換) / `comp`(構成部材)。
ツリーは hier、スイムレーンは time、グラフは全種を色分け表示。

## 再生成
```bash
# データ(graph.json)を編集したら:
python3 scripts/sync.py     # 全ファイルを再生成
```
個別に:
```bash
python3 scripts/gen_svg.py            # 旗艦SVG
python3 scripts/gen_components_ref.py # 構成部材リファレンス
python3 scripts/gen_matrix.py         # 価格・供給マトリクス
python3 scripts/gen_companies.py      # 銘柄マップ
python3 scripts/gen_l3.py             # L3ページ
python3 scripts/gen_cards.py          # 全カード
python3 scripts/gen_views.py          # 3ビュー
python3 scripts/gen_aux.py            # トレンド・スタブ・用語集
python3 scripts/gen_index.py          # トップ
```

## 認識論タグ
本文中で主張の種別を明示: `fact`(確立事実) / `infer`(推論) / `spec`(予測・不確実) / `stance`(見解)。
特に2030予測やガラス商用化時期は情報源で割れているため `spec` で明示的にヘッジ。

## 追補 (v1.6): 価格決定力 × 供給逼迫の2軸
- 全ノードに **価格決定力(pricing)** と **供給逼迫度(tightness)** の0-5軸を追加。各ノードに根拠ノートと確度タグ(fact/infer/spec)を併記。主要部材(HBM/CoWoS/ABF/MLCC/CCL/先端ノード)は2026実市況を検索で確認、他は寡占度・代替性・BOM比率から推論。
- 専用ページ **matrix.html(価格・供給マトリクス)**: 2軸散布図(右上=値上げ余地×逼迫が両高)+ 上位ランキング表 + 「差分で見る逼迫」(汎用 vs AI用ハイエンドの需給逆転)。
- 結論: 純粋な独占×完売の三役者(HBM/CoWoS/ABFフィルム)が突出。GPU本体は価格決定力最強だが利益を上流に吸われる。投資妙味の純度は上流独占部材が高い。
- 各カード(技術・部材)にも2軸と根拠ノートを表示。
- 55ノード / HTML76ページ。版: 1.7-nand。

Retimer / Redriver

Signal Conditioning IC — 高速シリアル信号の延伸・再生成

銅配線は周波数の平方根で損失が増え、PCIe Gen6 (64 GT/s NRZ→PAM4) / CXL 3.x / 224G PAM4 SerDes / UALink世代では、わずか数十cm〜1mのケーブル+コネクタでアイがほぼ閉じる。これを再生する半導体がRetimer/Redriverで、AIラック内では「銅配線を物理的に延ばす唯一の方法」として必須化した。Retimerは内部でCDR(Clock Data Recovery)を持ち信号をフルに再生するためジッタも初期化されるが、Redriverは増幅+等化のみで非同期遅延しか加えず低レイテンシ。AI scale-out拡張(L7)・scale-up(L6 NVLink/UALink)・PCIe/CXLいずれでも複数台搭載される。Astera Labsを筆頭に、224G PAM4の物理層複雑化と相まって「AI接続半導体」の代表として急成長中。

役割高速信号の等化・増幅・CDR再生
区分信号部品 / アナログミックスドシグナル
主配置CPU/GPU基板上、ライザー、ケーブル両端、AECモジュール、NIC内

RetimerとRedriverの違い

同じ「信号補正」でも内部構造と性能が大きく異なる。AI設計者の最初の選択は「銅で何mまで引きたいか・レイテンシ予算は何ns空いているか」で決まる。

軸Redriver (Linear)Retimer (Clocked)
機能受信信号を等化(CTLE)+増幅。CDR無し受信→CDR→FFE/DFE等化→PAM4デコード→PAM4エンコード→送信
ジッタ入力ジッタを通過させ、自身も付加CDRで一旦初期化、リファレンスクロック品質に再依存
BER等化のみで~10⁻¹²迄FEC前 BER 10⁻⁴→FEC後 10⁻¹⁵級まで改善
レイテンシ~ps〜数百ps(透過的)数十ns(クロックドメイン渡し)
消費電力 (224G/lane)~0.5W/lane級~1.5-3W/lane級
到達距離 (典型)~30-50cm銅 (224G PAM4)~1-2m銅+コネクタ (224G PAM4)
主用途PCIe Gen5/Gen6短距離、AOC内蔵PCIe Gen6長距離、CXL fabric、AEC、UALink/NVLink延伸
典型価格(2026)$5-15/ch級$30-100/ch級(16chで$500-1,500)

確立事実 PCIe Gen6 (64 GT/s PAM4) でケーブル+2コネクタ環境では、エンドポイント手前にRetimerが1〜2段必須とされる(PCI-SIG ECN/ガイドライン)。Gen5でもサーバボード長によっては1段挿入が標準化済。

銅とリンク予算 — なぜRetimerが要るのか

SerDesのリンク予算(Insertion Loss budget)は世代ごとに削減が困難になり、銅の到達距離は数十%ずつ短くなる。224G PAM4ではアイ高さがNRZ比1/3に圧縮されるため、ジッタとIL(挿入損失)の双方が世代1.5〜2倍厳しい予算で運用される。

世代Nyquist周波数典型IL予算銅PCB+ケーブル概算
PCIe Gen5 (32 GT/s NRZ)16 GHz~36 dB低損失Megtron系PCBで~50cm
PCIe Gen6 (64 GT/s PAM4)16 GHz (PAM4)~32 dB (アイ厳しい)同条件で~30cm、Retimerで延伸
112G PAM428 GHz~28 dBMegtron 6/7で~40cm+SMA
224G PAM456 GHz~26 dB (議論中、20 dB派も)同条件で~15-20cm、Retimer/AEC必須

確立事実 102.4T級スイッチ(Broadcom Tomahawk 6)のフロントポートからCageまでの基板配線でさえ、低損失材+多層+Retimer の組合せが必須化しており、ASIC直結DAC配線時代の終焉が始まっている。推論 これがCPO(共パッケージ光)推進の最大の経済的動機: 各ポートに Retimer + Optical Transceiver を載せるか、CPO で Retimer ごと光化するかの選択。

AEC (Active Electrical Cable) — Retimerが内蔵された銅ケーブル

Retimer/Redriver が DAC ケーブルのコネクタ内に組み込まれた製品。DAC(Direct Attach Copper)で2-3mまで を、AEC で5-7m延長できる(800G/1.6Tで)。光AOCより安価で消費電力が半分以下、ToR-Server間配線で主流化中。

分類典型距離消費電力(800G)用途
Passive DAC~2-3m~0W (passive)同ラック内ToR↔Server
ACC (Redriver内蔵)~3-4m~3-5WToR↔Server延伸
AEC (Retimer内蔵)~5-7m~8-10W隣接ラック・MoR配置
AOC (光)~30-100m~14-18Wラック間・Spine接続

確立事実 Credo Technology はAECで世界シェア最大級(Microsoft Azure / Meta の標準採用)、2025-2026年AI scale-out向けに売上が前年比+150%超。Astera Labsも Aries Smart Cable Module で参入し、Marvell・Broadcomもラインナップ。

確立事実 PCIe Gen6 Retimer量産能力は世界で実質Astera / Broadcom / Marvell の3社+補完のParade/Microchip。SerDes IP・DSPアーキ・CDR設計の難度が高く、新規参入は5-7年遅れ。推論 NVIDIAがAstera Labs を実質的に支援(IPO支援含む)している構造から、Retimer は「GPU売上に比例して必ず売れる消費財」として、AI接続セグメントで最も粗利が安定的な領域の一つ。

競合・代替の経路

選択肢長所短所影響時期
低損失PCB材料 (Megtron 7/8、PTFE系)Retimer段数削減材料コスト2-3倍、配線距離は依然限界2026以降のサーバボードで標準化中
同軸ツイナックスケーブル直結 (NVL72 Mid-Plane)レイテンシ・電力最良2m以内、ラック内専用NVL72/Rubinラックで採用済
CPO (共パッケージ光)ポート毎の銅延長を不要化、レイテンシ低減歩留・熱・保守性、量産2027-scale-up先行、scale-outは2027以降
LPO (Linear Pluggable Optics)Retimer/DSP無しでも光モジュール動作距離・温度マージンが厳しい2026に一部採用、汎用化はリスク
LRO (Linear Receive Optics)送信DSP+Retimer無し受信側DSPはあり、設計難2026-2027検討段階

推論 LPO/CPO は「Retimer不要化」ロジックを持つが、CPO はASICパッケージ側のSerDes/DSP統合に置換するだけでDSP半導体としては依然必要。光が増えてもPCIe/CXL/NVLink/UALink銅延長は残り、Retimer需要は2030年まで毎年二桁成長予測(LightCounting / 650 Group)。

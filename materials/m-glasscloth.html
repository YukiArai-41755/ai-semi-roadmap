T-glass / ガラスクロス

Low-CTE / Low-Dk Glass Cloth (T-glass / NE-glass / Q-glass)

IC基板の有機コア(CCL)に織り込む低CTE・低Dkガラス繊維布。樹脂単独では機械強度・熱膨張・寸法安定性が足りないため、ガラス繊維を経糸/緯糸で平織りしたクロスを補強材として含浸する。AI server基板は120mm角級まで大型化し、層数13+13(20-26層)、リフロー250°C級、液冷時の熱衝撃、224G信号の低Df化を同時要求するため、ガラスクロスの選定がパッケージの反り・歩留り・電気特性の三軸で律速する。Nittobo(日東紡)のT-glass・NE-glassが世界シェア90%級を独占し、Apple/NVIDIA/Google/Amazonの基板調達担当が直接交渉するレベルの逼迫=2026年AI半導体最大級のボトルネック。緩和は2027後半。演算もしない地味な素材だが、独占×完売×置換不可の純度はABFフィルム級。

役割CCL補強材(寸法安定・低CTE・電気特性付与)
区分基板素材(繊維材料)
主配置CCLコア層内、樹脂と織り混ざる形で全層に

ガラスクロスの種類と化学組成

ガラスは組成でE-glass(電気用標準)、T-glass(低CTE)、NE-glass(低Dk)、Q-glass(石英) 等に分類される。AI server基板の進化は E-glass → T-glass → NE-glass → Q-glass の方向。

種類主成分CTE (ppm/K)Dk @ 10GHz用途
E-glass(電気用)SiO2 55% / CaO 22% / Al2O3 14%5.56.2-6.6FR-4・標準。AI用途は不可
T-glass(高ヤング率・低CTE)SiO2 65% / Al2O3 23% / MgO 11%3.05.3-5.7AIパッケージ基板コアの主流
NE-glass(低Dk)SiO2 52% / B2O3 20% / Al2O3 12% / アルカリ等3.44.4-4.6低損失PCB・1.6T サーバマザーボード
L-glass / 低誘電(他社版)類似低Dk組成3.0-3.54.5-4.8AGCや旭化成の対応グレード
Q-glass(石英)SiO2 ~99.99%0.53.82.5D/3D・CPO周辺で試作。高価・加工難

確立事実 NittoboのT-glassは独自製法で1980年代から確立、AI server向けは特に1000番手(NT-1080)・1078番手等のシリーズ。NE-glassは1990年代に上市された低Dk版で、Megtron 7N/8系のCCLには必須。Q-glass(石英)は石英繊維の極細紡績難度が高く、量産材としては未確立。

織物構造とパラメータ

ガラスクロスは経糸・緯糸の本数(糸種)、ヤーンの本数(BCM/番手)、織り密度(本/inch)で性能が決まる。AI用は薄物(1015 / 1027 / 1037 / 1067 / 1078 / 1080番手)を多数層使い、樹脂含浸率を上げて誘電均一性とレーザビア加工性を両立させる。

番手(代表)厚み(μm)使われ方
1080~50標準PCB
1078~40サーバ高速ボード
1067~32AI高速ボード
1037 / 1027 / 1017~24-15AIパッケージ基板コア・低Df層
1015 / 1010~13-10AI先端・薄物・SAP対応

確立事実 薄物クロス(1015以下)はナノレベル欠陥が許されず、織機・後加工(オープニング処理・カップリング処理)の精度が極端に高い。新規参入は織機投資+ガラスマーブル生産+表面処理+量産安定の積み重ねで5-7年の参入障壁。

製造プロセス

(1) ガラスマーブルを電気炉で溶融 → (2) ブッシングで連続繊維紡糸(直径 5-9 μm) → (3) ヤーン化 → (4) 平織り → (5) 脱バインダ(熱処理で集束剤を除去) → (6) ヒートクリーニング → (7) シランカップリング処理(樹脂との接着性付与) → (8) 検査・スリッタ。原料ガラス組成 + 連続繊維紡糸技術 + 表面処理の三層垂直統合がNittoboの競争源泉。

原料石英・アルミナ・ボーラックス・マグネシア等(高純度・低アルカリ)
繊維径5-9 μm (AI先端は 5-6 μm 級)

確立事実 Nittoboの寡占はガラス組成特許+織機ノウハウ+表面処理+顧客認証(MGC BT樹脂 / Megtron系等とのレシピ統合)の積み重ね。推論 AGC・旭化成・Nan YaがNE-glass同等品で食い込めても、T-glass(高ヤング率系)で日東紡を置換するには5-7年とみられる。最終ユーザが直接契約する動きは「Substrate-as-a-Strategic-Asset」化の象徴。

ABF

Ajinomoto Build-up Film — 味の素ビルドアップフィルム

有機パッケージ基板のコア(CCL)の上下に積層するエポキシ系絶縁フィルム。これを多層積み上げ、レーザ穴あけ→デスミア→Cuめっきを繰り返してダイ-基板間の微細再配線層(RDL)を形成する。AI GPU/CPU/HBMパッケージは2.5D/3D化でI/O数が10万本級まで急増し、配線層数は11+11層 → 13+13層 → 15+15層へと膨らんでいる。1枚のフィルムが薄く・無欠陥・厚みばらつき小さく・誘電損失低くないと、224G PAM4信号は到達しない。味の素ファインテクノが世界シェア95%級を独占し、AI server BOMの中で純度最大の独占×完売×値上げ主導を成立させている戦略部材。

役割ビルドアップ層間絶縁フィルム / RDL層形成
区分構造・配線基盤 (有機絶縁材料)
主配置CCLコアの上下 / 各ビルドアップ層間 / FCBGA・FCCSPの全層

材料系と化学構造

ABFはエポキシ樹脂・硬化剤・球状シリカフィラ・難燃剤を含むBステージ(半硬化)樹脂を、PETキャリアフィルム上に薄く塗工したドライフィルム型ビルドアップ材。1989年に味の素中央研究所で「うま味調味料の副産物アミノ酸の応用」から派生した経緯を持つ。RCCやプリプレグと違いガラスクロスを含まないため、薄層化と微細レーザビア加工性で先行した。

有機基板内での階層的役割

FCBGA基板は典型的に「ビルドアップ層 (上) / CCLコア / ビルドアップ層 (下)」のサンドイッチ構造。AIパッケージでは上層がダイとの細線インターフェース、下層がBGAボールへの中継。ABFはこの「ビルドアップ層」を構成する全フィルム=パッケージ内の大部分の層数を担う。

製造プロセスとABFの役割

典型工程: (1) CCLコアにCu配線パターン形成 → (2) ABFを真空ラミネート → (3) 熱硬化 → (4) CO2/UVレーザでビア穴形成 → (5) デスミア (過マンガン酸塩でビア底樹脂除去) → (6) パラジウム触媒 + 無電解Cuめっき → (7) ドライフィルムレジスト + 電解Cuめっき + エッチング → (8) これを繰り返して所望層数。各層の厚みばらつき・反り・微細パターン解像性が全て後工程に影響する。

推奨配線L/S7/7 μm (主流) → 5/5 μm (Rubin世代) → 2/2 μm (RDL先端品)

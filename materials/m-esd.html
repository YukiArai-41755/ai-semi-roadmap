ESD/TVS保護素子

ESD Protection Diode / TVS (Transient Voltage Suppressor) Diode

静電気放電(ESD)、サージ、過渡電圧から高速I/Oと電源を守る半導体保護素子。AI server は外部接続点(PCIe Gen6、800G/1.6T Ethernet、CXL、NVLink、QSFP-DD光モジュール、管理用USB/UART、電源コネクタ等)が数十〜数百ポート単位で並び、ESD保護はリンク信頼性の暗黙の前提。一方、224G PAM4のような56GHz級信号では保護素子の寄生容量(Cj < 0.1 pF級が必須)が信号品質を直接劣化させるため、ただの「汎用ダイオード」では足りない。低容量・低クランプ電圧・高サージ耐量を同時要求するRF級ESD素子のシェアが拡大し、Nexperia、onsemi、Littelfuse、STMicro、Bourns、Toshiba、Diodes、Infineonが寡占。AI server BOMの中では金額シェアは小さいが、L6/L7ネットワークの信頼性を支える縁の下の部材。

役割ESD・サージ・過渡電圧保護
区分保護部品(半導体ダイオード)
主配置高速I/Oコネクタ近傍、電源入力、リセット/管理ライン

確立事実 800G/1.6T光モジュールのQSFP-DD/OSFPコネクタ周辺はCj < 0.1pF級のRF ESD素子(例: Nexperia PESD0402-100、ON Semi ESD7104MUTAG、STMicro ESDA8P25-1U1M等)が必須。AI Spineスイッチ1台あたりこれら高速ポートが32-128個並び、ポートごとに差動2本 = 64-256個の保護素子が並ぶ。

CCL

Copper Clad Laminate — 銅張積層板 (パッケージ基板コア)

ガラスクロスにエポキシ/BT/PPE/PTFE系樹脂を含浸させ、両面に銅箔を貼って熱プレス硬化したパッケージ基板の機械的な芯。CCLの上下にABFを積み上げてビルドアップ層を形成し、その両側に表裏のRDLが付く。AI server基板は大型化(120mm角級)・厚膜化(層数増)・高Tg化・低Dk/Df化が同時進行し、CCLは「ただの絶縁板」から反り・誘電・熱の三重設計対象へと変質した。Resonac(旧昭和電工)、Mitsubishi Gas Chemical (BT樹脂)、Panasonic、Doosan、ITEQ、EMC、Nan Yaが主軸。BTコア大型品では実質3-4社の寡占で、+30%値上げの主導力を持つ。

役割パッケージ基板コア(機械的芯+一次配線層)
区分構造・配線基盤 (有機積層板)
主配置FCBGA/FCCSP基板の中心、ビルドアップ層に挟まれた位置

CCLの構造とBOM

典型CCL = ガラスクロス(補強材)+樹脂(マトリクス)+銅箔(導体) のサンドイッチ。各構成材で材料系・グレードが異なる。AI基板用は低Df樹脂 + T-glassクロス + HVLP/HVLP-2銅箔の組み合わせが定番。

補強材 (Reinforcement)T-glass / NE-glass / E-glass / Q-glass 等、ガラスクロス
樹脂 (Resin)BT (Bismaleimide-Triazine) / 改質エポキシ / PPE / 低Dkハイブリッド / PTFE系
導体 (Conductor)HVLP / HVLP-2 / HVLP-3 銅箔 (2-12 μm厚)

確立事実 AI server向けCCLはパッケージ基板コア(BT系 MGC主導)とPCB低損失系(Panasonic/Resonac/ITEQ/EMC主導)で主役が異なる。MLCCのような単一寡占ではないが、低損失系で実質4-5社。推論 中国Shengyi等は2026時点でAI Tier-1認証が一部進行中だが、信頼性・歩留り・量産安定で日台勢にやや劣後。

代替・競合技術

代替特性状況
ガラスコア基板CTE 3-9 ppm/K, 反り極小, 高剛性Intel Clearwater Forest世代から本格採用、Samsung・AGC・Absolicsが推進。有機CCLを根本から置換するシナリオ

コアレス基板CCLを省略、薄・軽モバイルAP・小型FCCSPで採用。サーバ大型パッケージは反りで不可

推論 ガラスコアは2027-2030でAI先端パッケージ基板の有力対抗。有機CCLが消えるのではなく、コアの一部がガラスへ置換、ビルドアップ層には引き続きCCL周辺技術(ABF)が使われると見られる。CCL全体の量は減らない。
